O trabalho está organizado da seguinte forma:
- na pasta text_files tem o código que cria ficheiros aleatóriamente dos tamanhos pedidos e os respetivos ficheiros.
- na pasta plots estão todas as tabelas geradas pelos códigos.
- fora das pastas tem todos os códigos dos diferentes mecanismos criptográficos, dos seus gráficos individuais, dos seus gráficos de comparação e o main (pensamos que seria benéfico ter um para ser mais fácil correr tudo na totalidade depois de testados os códigos individualmente)

Como correr?
- se for desejado criar novos ficheiros de texto aleatórios correr diretamente o ficheiro create_files.py qeu se encontra na pasta text_files
- se for desejado correr todos os outros códigos, incluindo criação de gráficos para mostrar os resultados, correr o ficheiro main.py. os novos gráficos criados vão se encontrar na pasta plots.

Bibliotecas necessárias de instalar:
- matplotlib
- cryptography
